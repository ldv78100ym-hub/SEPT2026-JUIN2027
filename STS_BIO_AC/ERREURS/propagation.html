🔁 5. Propagation des erreursThéorie

Définition : Transmission des erreurs des grandeurs mesurées vers le résultat final.

5.1 Opérations de base

Addition / Soustraction$$ \Delta (x \pm y) = \Delta x + \Delta y $$
Multiplication / Division$$ \frac{\Delta (x \times y)}{|x \times y|} = \frac{\Delta x}{|x|} + \frac{\Delta y}{|y|} $$
Puissance$$ \frac{\Delta (x^n)}{|x^n|} = n \times \frac{\Delta x}{|x|} $$

⚠️ Attention à l’incertitude relative d’une somme ou d’une différence

La formule \(\Delta (x \pm y) = \Delta x + \Delta y\) donne l’incertitude absolue du résultat. Pour obtenir l’incertitude relative de ce résultat, il faut diviser cette incertitude absolue par la valeur absolue du résultat :

$$ \text{Incertitude relative sur } (x \pm y) \;=\; \frac{\Delta x + \Delta y}{|x \pm y|} $$

Conséquence cruciale : si la différence \(|x - y|\) est petite (par exemple \(x = 100,0\) et \(y = 99,0\)), l’incertitude relative peut devenir énorme. Avec \(\Delta x = \Delta y = 0,5\), l’incertitude absolue est \(1,0\), mais le résultat est \(1,0\) → incertitude relative = \(1,0 / 1,0 = 100\) % ! C’est pourquoi il faut éviter de soustraire deux grands nombres très proches en laboratoire.

📌 Exemple : une balance tarée puis une deuxième pesée peut introduire une énorme incertitude si les deux masses sont voisines.

5.2 Dérivées partielles (cas général)

Pour $f(x_1, \ldots, x_n)$$$ \Delta f = \sqrt{ \left( \frac{\partial f}{\partial x_1} \Delta x_1 \right)^2 + \cdots + \left( \frac{\partial f}{\partial x_n} \Delta x_n \right)^2 } $$

🔬 Exemple détaillé : volume d’une pipette cylindrique

On considère un cylindre de rayon \(r\) et de hauteur \(h\) : \[ V = \pi \, r^2 \, h \] Les mesures sont : \(r = 0,50 \pm 0,01\ \text{cm}\) et \(h = 10,0 \pm 0,1\ \text{cm}\).

Étape 1 – Calcul des dérivées partielles

La fonction \(V\) dépend de deux variables \(r\) et \(h\). On calcule les dérivées partielles : \[ \frac{\partial V}{\partial r} = 2\pi\, r\, h \qquad \text{et} \qquad \frac{\partial V}{\partial h} = \pi\, r^2 \] Explication : pour \(\dfrac{\partial V}{\partial r}\), on dérive \(\pi r^2h\) par rapport à \(r\) (\(r\) est dans ce cas la variable, h est considérée comme une constante) : on obtient \(2\pi rh\) ;
pour \(\dfrac{\partial V}{\partial h}\), on dérive \(\pi r^2h\) par rapport à \(h\) (\(h\) est dans ce cas la variable, r est considérée comme une constante) : on obtient \(\pi r^2\) ;

Étape 2 – Application de la formule de propagation

On combine les contributions en quadrature (car les erreurs sont supposées indépendantes) : \[ \Delta V = \sqrt{ \left( \frac{\partial V}{\partial r} \, \Delta r \right)^2 + \left( \frac{\partial V}{\partial h} \, \Delta h \right)^2 } = \sqrt{ \left( 2\pi r h \, \Delta r \right)^2 + \left( \pi r^2 \, \Delta h \right)^2 } \]

Étape 3 – Passage à l’incertitude relative

On divise par \(V = \pi r^2 h\) pour obtenir : \[ \frac{\Delta V}{V} = \sqrt{ \left( \frac{2\pi r h \, \Delta r}{\pi r^2 h} \right)^2 + \left( \frac{\pi r^2 \, \Delta h}{\pi r^2 h} \right)^2 } = \sqrt{ \left( 2\,\frac{\Delta r}{r} \right)^2 + \left( \frac{\Delta h}{h} \right)^2 } \] On retrouve bien la règle : le facteur \(2\) vient de l’exposant de \(r\) dans la formule.

Étape 4 – Application numérique

\[ \frac{\Delta r}{r} = \frac{0,01}{0,50} = 0,02 \quad \Rightarrow \quad 2 \times 0,02 = 0,04 \] \[ \frac{\Delta h}{h} = \frac{0,1}{10,0} = 0,01 \] On combine : \[ \frac{\Delta V}{V} = \sqrt{ (0,04)^2 + (0,01)^2 } = \sqrt{ 0,0016 + 0,0001 } = \sqrt{ 0,0017 } \approx 0,041 \] soit une incertitude relative d’environ 4,1 %.

Étape 5 – Incertitude absolue (facultative)

On calcule d’abord \(V = \pi \times (0,50)^2 \times 10,0 \approx 7,85\ \text{cm}^3\). Puis \(\Delta V = V \times 0,041 \approx 0,32\ \text{cm}^3\). Le résultat final s’écrit donc \(V = 7,85 \pm 0,32\ \text{cm}^3\).
